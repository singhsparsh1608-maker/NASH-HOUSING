#  Nashville Housing Data Analysis

This project explores the **Nashville Housing Dataset** using **SQL** and **Python** to clean messy data, analyze housing trends, and present insights through visualizations. It was built to strengthen my skills in data cleaning, SQL querying, exploratory data analysis (EDA), and dashboard development.

##  What I Did

* Cleaned and transformed raw housing data using SQL.
* Removed duplicates and handled missing values.
* Standardized dates and split address fields for better analysis.
* Explored housing prices, sales trends, land use, and property characteristics.
* Built visualizations and an interactive dashboard using Python and Streamlit.
